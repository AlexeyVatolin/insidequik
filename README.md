# INSIDE QUIK

## Описание проекта

Есть главный счет (счет инвестора, хозяина), деньги которого распределяются между каждым трейдером. Трейдеры торгуют на бирже параллельно с главного счета. Хозяин не только видит все действия, но хочет знать действия каждого трейдера по отдельности (какие акции покупал, сколько денег у него осталось, насколько правильно и успешно он ими распоряжается и т.д.). 

Наше приложение позволит трейдеру через приложение или веб интерфейс заходить в систему под своей учетной записью и совершать операции, которые будут отправляться на сервер, где будет установлена программа QUIK. Терминал QUIK будет обрабатывать запросы и возвращать результаты запросов трейдеру. Все действия конкретного трейдера также отправляются на сервер, где с помощью админской учетной записи уже можно будет просмотреть клиентские данные. 

### Почему именно QUIK?

Мы выяснили, что 90% трейдеров в России знают и пользуются этим терминалом, но их не устраивает сложный и старомодный интерфейс. Также выяснили, что инвесторы хотели бы знать, что делают их трейдеры. Аналогов нашего приложения пока что нет. Таким образом, мы считаем, что разработка данного приложения была бы очень актуальна. Как показывает статистика, большинство трейдеров доверяют квику и хотят, чтобы их операции совершались через него, но с более понятным интерфейсом и улучшенным функционалом.

## Как начать работать?

1. [Открыть демо-счет](https://broker.ru/demo?refid=7050&utm_source=yandex&utm_medium=cpc&utm_campaign=demo_rf&yagla=11730752&yclid=666555300110148004) у БКС брокера.  
2. Далее на почту, указанную при открытии демо-счета, придут всё необходимое для доступа к демо-счету (логин, пароль, сама программа QUIK, ключи для демо-доступа, инстуркция по установке QUIK).  
3. Выполнить все действия, указанные в инструкции.
4. Далее в окне программы (сверху) `Сервисы -> Lua скрипты -> Добавить`.
5. Выбрать файл в папке проекта `\insidequik\src\QuikSharp` и запустить его.
